📖 Panduan Penggunaan
🚀 Mulai Cepat — 4 Langkah
⚙️
1. Setup Perusahaan
Klik tombol "⚙️ Pengaturan" di atas. Isi nama toko/perusahaan dan upload logo.
👥
2. Tambah Karyawan
Menu Karyawan → tambah manual satu per satu, atau upload file Excel/CSV sekaligus.
📅
3. Catat Absensi
Menu Absensi → klik kotak per karyawan per hari, atau absen cepat dari Dashboard.
📄
4. Generate Slip Gaji
Menu Slip Gaji → pilih karyawan & periode → print/PDF langsung dengan logo perusahaan.
📅 Cara Mengisi Absensi
Klik kotak pada tabel untuk memilih status. Setiap klik akan berganti otomatis:
H = Hadir I = Izin S = Sakit A = Alpha L = Libur
💡 Tips: Dari Dashboard, gunakan tombol absen cepat (H/I/S/A/L) untuk mencatat semua karyawan sekaligus dalam satu klik.

📥 Klik "Export Rekap" untuk cetak rekap absensi bulanan.
📄 Cara Membuat Slip Gaji
  1. Buka menu Slip Gaji
  2. Pilih nama karyawan dari dropdown
  3. Data gaji otomatis terisi dari profil karyawan
  4. Pilih periode bulan yang akan dibuat
  5. Sesuaikan komponen gaji jika perlu (lembur, potongan, dll)
  6. Klik "🖨️ Generate & Print Slip"
  7. Pilih Print atau Save as PDF
💡 Logo dan nama perusahaan akan otomatis muncul di slip gaji sesuai pengaturan perusahaan.
📤 Upload Data Karyawan via Excel
  1. Klik "📥 Download Template" di menu Karyawan
  2. Buka file di Excel atau Google Sheets
  3. Isi data karyawan mulai baris ke-5
  4. Simpan sebagai CSV (File → Save As → CSV)
  5. Klik "📤 Upload Excel/CSV"
  6. Cek preview data, lalu klik "✅ Import Semua"
⚠️ Untuk file .xlsx, simpan dulu sebagai CSV agar bisa dibaca dengan sempurna.
💾 Tentang Data & Penyimpanan
✅ Semua data tersimpan di browser (localStorage) — tidak perlu internet setelah file dibuka.
✅ Data aman — tidak dikirim ke server manapun, hanya ada di perangkat kamu.
⚠️ Gunakan browser yang sama — data tidak berpindah jika buka di browser berbeda.
⚠️ Jangan clear browser data — ini akan menghapus semua data yang tersimpan.
💡 Tips backup: Rutin export rekap absensi & laporan penggajian sebagai PDF setiap bulan.
🧾 Cara Menggunakan Laporan Penggajian
Laporan otomatis berisi:
  • Total karyawan yang digaji
  • Total pengeluaran gaji bulan tersebut
  • Rata-rata gaji per karyawan
  • Detail per karyawan (gaji pokok, tunjangan, lembur, potongan, bersih)
Tips penggunaan:
  • Pilih bulan menggunakan selector di kanan atas
  • Laporan terisi otomatis dari data slip gaji yang sudah dibuat
  • Klik "🖨️ Print Laporan" untuk cetak atau save PDF
  • Simpan laporan tiap bulan sebagai arsip penggajian
